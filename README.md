#Pattern Generator and Number Analyzer(Logic Box)

This is a Python-based menu_driven program that allows users to generate geometric star patterns and analyze numerical properties within a specific range.

##Features:
-**Pattern Generator:** Creates a right-angled triangle using nested loops based on user-defined rows.
-**Number Analyzer:** Iterates through a specified range of numbers, identifies whether each is Odd or Even, and calculates the total sum of the range.
-**Input Validation:** Ensures inputs are valid numbers and provides error feedback.

##Assumptions:
- The use will enter integers for the range selections.
- The right-angled triangle is generated using standards asterisks('*').
