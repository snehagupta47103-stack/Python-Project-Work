# Personal Data Collector

This simple Python script gathers basic personal information from the user, displays it along with its data type and memory address, and performs a couple of basic calculations.

## Features

*   Collects user's name, age, height, and favorite number.
*   Determines and displays the data type and memory ID for each input.
*   Calculates an approximate birth year based on the entered age.
*   Demonstrates float to integer conversion (with rounding behavior).

## How to Run

1.  Ensure you have Python installed.
2.  Save the code as a `.py` file (e.g., `data_collector.py`).
3.  Run the script from your terminal:
    ```bash
    python data_collector.py
    ```
