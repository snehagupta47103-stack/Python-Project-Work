# Employee Management System

## Project Overview

Employee Management System is a menu-driven Python application developed using Object-Oriented Programming (OOP) concepts. The system allows users to create and manage Person, Employee, and Manager records and display their information through an interactive menu.

The project demonstrates important OOP concepts such as inheritance, encapsulation, method overriding, constructors, destructors, getter/setter methods, and the use of `super()` and `issubclass()`.

---

## Features

* Create Person records
* Create Employee records
* Create Manager records
* Display Person details
* Display Employee details
* Display Manager details
* Menu-driven interface
* Encapsulation using private attributes
* Getter and Setter methods
* Inheritance and Method Overriding

---

## OOP Concepts Implemented

### 1. Constructor

Used to initialize object attributes.

### 2. Destructor

Used for object cleanup.

### 3. Encapsulation

Employee ID and Salary are stored as private attributes.

### 4. Getter and Setter Methods

Used to access and modify private data members.

### 5. Inheritance

* Employee inherits from Person
* Manager inherits from Employee

### 6. Method Overriding

The `display()` method is overridden in the Manager class.

### 7. super()

Used to access parent class constructors and methods.

### 8. issubclass()

Used to verify inheritance relationships.
