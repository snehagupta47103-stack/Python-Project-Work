# 🎓 Student Management System (Python CLI Project)

A simple yet powerful **Student Management System** built using **Python**.  
This project allows users to manage student records directly from the command line using a menu-driven interface.

---

# ✨ Features

✅ Add New Students  
✅ Display All Student Records  
✅ Update Student Information  
✅ Delete Student Records  
✅ Display Subjects Offered  
✅ Interactive Menu System  
✅ Beginner-Friendly Python Project  

---

# 📌 Project Preview

```bash
Select an option:
1. Add Students
2. Display All Students
3. Update Student Information
4. Delete Students
5. Display Subjects Offered
6. Exit
````

# 🚀 How to Run the Project
## 📥 Step 1: Clone the Repository

```bash
git clone https://github.com/your-username/student-management-system.git
```

---

## 📂 Step 2: Open Project Folder

```bash
cd student-management-system
```

---

## ▶️ Step 3: Run the Program

### On Windows

```bash
python main.py
```

### On Mac/Linux

```bash
python3 main.py
```

# 📖 Functionalities Explained

## ➕ Add Student

Users can add:

* Student ID
* Name
* Age
* Grade
* Date of Birth
* Subjects

---

## 📋 Display Students

Displays all saved student records in a clean format.

Example:

```bash
Student ID: 101 | Name: Ali | Age: 18 | Grade: A
```

---

## ✏️ Update Student Information

Update:

* Name
* Age
* Grade
* Subjects

using Student ID.

---

## ❌ Delete Student

Remove a student record permanently using Student ID.

---

## 📚 Subjects Offered

Displays available subjects:

* Mathematics
* Science
* English
* Computer Studies
* History
* Accounts
* Statistics
