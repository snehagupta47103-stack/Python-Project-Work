# Array Operations and Python Functions Demonstration

## 📌 Project Overview

This is a menu-driven Python program designed to demonstrate important Python programming concepts while performing basic array operations. The project allows users to work with both 1D and 2D arrays and showcases features such as:

* Functions
* Global Variables
* Recursion
* Lambda Functions
* `*args` and `**kwargs`
* Multiple Return Values
* Sorting
* Function Documentation Strings (`__doc__`)

The program is suitable for beginners learning Python fundamentals and function-based programming.

---

## 🚀 Features

### 1. Input 1D Array

* Accepts integer values from the user.
* Stores them in a global list.
* Generates a summary dictionary.

### 2. Input 2D Array

* Accepts multiple rows of integer values.
* Stores them as a nested list (matrix).

### 3. Display Summary

Shows:

* Length of array
* Sum of elements
* Minimum value
* Maximum value

### 4. `*args` Demonstration

* Passes array elements as variable-length arguments.

### 5. `**kwargs` Demonstration

* Displays key-value pairs using keyword arguments.

### 6. Factorial Using Recursion

* Calculates factorial of a number recursively.

### 7. Lambda Function Demo

* Uses a lambda expression to calculate the square of a number.

### 8. Global Variable Demo

* Displays the global summary dictionary.

### 9. Multiple Return Values

* Returns and displays minimum and maximum values from the array.

### 10. Sorting

* Sorts the 1D array.
* Sorts the rows of the 2D array.

### 11. Display 2D Array

* Prints all rows of the matrix.

### 12. Documentation Viewer

* Displays docstrings of implemented functions.
